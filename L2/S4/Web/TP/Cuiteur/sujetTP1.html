TP 1 : HTML - Mise en forme et mise en page simple

Ce premier TP a pour objectif de vous familiariser avec les bases du langage HTML et les bases de la mise en page avec les CSS.

Pré-requis

Ce TP couvre les parties suivantes du cours HTML :
  • Présentation de HTML
  • Les feuilles de styles
  • CSS utilisation avancée
  • Vidéos HTML chapitre 1 - Principes de base
  • Vidéos HTML - Les feuilles de styles

Travail à réaliser

L'objectif est de réaliser 2 pages de l'application Cuiteur : une page de présentation et une page d'illustrations.

Il faut procéder par étapes (ou itérations) pour la conception de la page finale presentation.html . Placez chaque étape dans un fichier séparé pour bien voir chaque phase de construction et pouvoir revenir facilement en arrière quand vous vous êtes emmêlés les doigts dans le clavier.

Vous DEVEZ reproduire à l'identique les 5 pages suivantes qui consistuent des étapes de construction.

Remarque importante : vous n'êtes pas en train de faire une course de vitesse, mais la conception de pages qui doivent avoir un code clair et une mise en forme / page facilement identifiable et modifiable. Le but n'est pas la rapidité mais la qualité.

Etape 1

Réaliser la page presentation_1.html avec la structuration de son contenu textuel.

Commencez par une page HTML simple, contenant uniquement les textes (à télécharger ici) structurés par des tags HTML ayant la bonne signification sémantique. A cette étape l'utilisation de tags div est interdite. Vous n'avez pas le droit d'utiliser plus de 2 tags div pour réaliser la page complète.

Cette page n'a pas de feuille de styles.

Placez les liens sur les éléments qui en ont et testez les, même si certains pointeront sur des pages qui ne sont pas encore réalisées.

Régle à utiliser : KISS (Keep It Simple Stupid)

Etape 2

Réaliser la page presentation_2.html avec les images ajoutées aux textes de l'étape précédente.

Insérez les images (à télécharger ici.) dans la page précédente. Ne vous occupez pas pour l'instant du placement spécifique des images

Placez les liens sur les éléments qui en ont et testez les, même si certains pointeront sur des pages qui ne sont pas encore réalisées.

Etape 3

Réaliser la page presentation_3.html avec une feuille de styles pour la mise en forme des textes.

Ajoutez une feuille de styles ( presentation.css ). Dans un premier temps il faut définir la mise en forme : les styles des textes, titres, parapgraphes, gras, liens, etc.

Commencez par ré-initialiser la feuille de style par défaut du navigateur selon la méthode proposée par : http://meyerweb.com/eric/tools/css/reset/.

Définissez le bloc de contenu de la page avec un tag div et stylez le avec une image de fond et une dimension (rappel : vous n'avez pas le droit d'utiliser plus de 2 tags div pour réaliser la page complète).

Stylez les tags HTML nécessaires et définissez des classes pour faire de la mise en forme. Il ne faut pas essayer de vouloir tout faire à la fois. En commençant par la mise en forme vous simplifierez beaucoup la phase suivante qui est la mise en page.

Faites autant d'itérations que vous voulez pour terminer la mise en forme. Conseil : quand vous avez terminé une partie significative sauvegardez votre feuille de styles pour un retour en arrière facile. Normalement, à ce stage vous ne devriez plus modifier le code HTML, mais uniquement le code CSS.

Régle à utiliser : KISS (Keep It Simple Stupid)

Etape 4

Réaliser la page presentation.html terminée avec la feuille de styles complètée par la mise en page.

Vous êtes maintenant dans la dernière partie du travail : la mise en page. C'est la plus difficile car il faut utiliser les propriétés de style de positionnement. Certains éléments seront positionnés avec des classes définissant des blocs flottant, d'autres devront être positionnés de manière absolue pour respecter ce qui est demandé. A vous de trouver ...

C'est dans cette phase que vous aurez à utiliser le deuxième tag div permis.

La régle KISS est encore plus importante ici que dans les autres phases.

Etape 5

Réaliser la page moyens.html.

Cette page est très facile à réaliser. Comme il y a très peu de texte et que vous avez maintenant de l'entraînement, vous pouvez écrire tout le code HTML en une seule fois, dans faire de séparation phase texte / phase image. Les images doivent être taggées correctement car les liens de la page presentation.html arrive directement au bon endroit dans la page et pas systèmatiquement en haut de page.

Avant de vous occupez des styles, vous devez avoir une structure HTML simple (KISS) et solide.

Cette page utilise elle aussi uniquement 2 tags div : un pour le bandeau et un pour le contenu.

Le bandeau est particulier car il reste toujours placé en haut de la page et ne disparait pas quand on scrolle. Le fond du bandeau est fait avec une propriété CSS d'opacité.

Répertoires

Vous devez utiliser l'arborescence de répertoires suivante : 

cuiteur

   |      |___ photos
   |___ styles

Cette arborescence sera réutilisée tout au long du développement de l'application. Les noms doivent être respectés et écrit en minuscules.

Les pages presentation_1.html, presentation_2.html, presentation_3.html, presentation.html et moyens.html seront placées dans le répertoire html .

Les images et photos seront placées dans les répertoires images et images/photos . Les images que vous devez utiliser sont à télécharger ici.

Les feuilles de styles seront placées dans le répertoire styles .

Les répertoires php et upload seront utilisés plus tard.

Mise en forme / mise en page

Les polices à utiliser sont par ordre de préférence Verdana, Geneva, Sans-Serif. La taille par défaut est de 16 pixels avec une interligne de 1.6.

Les titres ont les tailles et marges suivantes :

Taille 20 pixels 26 pixels 36 pixels
Gras oui oui oui
Marges 18 pixels 22 pixels 26 pixels

Le plus gros titre est souligné par un trait de 3 pixels d'épaisseur.

couleur de fond couleur des textes couleur des liens couleur texte en bas
#20190F #E8DBB1 #3C96F1 #EF9C3B

Les liens sont en gras et se soulignent quand ils sont survolés par la souris.

Les liens "Cuiteur" pointent tous sur la page inscription.php qui se trouve dans le répertoire php. Cette page sera réalisée plus tard.

Le lien "microblogage" pointe sur la page Microblog de wikipedia. Cette page doit s'ouvrir dans une nouvelle fenêtre (ou onglet).

Les 6 liens des photos dans la partie "Les moyens techniques de Cuiteur" pointent sur les photos correspondantes en grande taille dans la page moyens.html. Quand on clique sur un de ces liens on doit arriver directement sur la photo correspondante.

La mise en page est très simple. Le contenu est mis dans un bloc de 800 pixels de large, centré dans la fenêtre du navigateur.

Les images sur les bords gauche et droit sont à cheval sur le bloc de contenu et dépasse à gauche ou à droite de 60 pixels.

Les textes doivent être placé à 32 pixels des bords grauche et droit du bloc contenu.
