dl : liste de définitions

    Les listes de définitions permettent d'associer deux contenus entre eux. Si le 1er exemple qui vient à l'esprit est un glossaire (des termes sont associés à leur définition), on peut aussi employer ce type de liste pour des FAQ (Foires Aux Questions) ou des traductions.

    Une liste de définitions se fait avec 2 élements HTML :

    • l'élément dl marque les limites de la liste,
    • l'élément dt définit un terme de la liste.
    • l'élément dd définit le contenu associé au terme.

    Si on peut faire une analogie avec le concept des tables de hashage (des association de couples nom/valeur), les listes de définitions permettent aussi de faire des associations multiples (un nom avec plusieurs valeurs ou plusieurs noms avec la même valeur).
