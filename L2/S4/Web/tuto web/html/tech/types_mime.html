Les types MIME

MIME est l'abréviation de "Multi-purpose Internet Mail Extensions".

C'est un standard qui a été proposé par les laboratoires Bell Communications en 1991 pour étendre les possibilités du courrier électronique en permettant d'insérer des documents (images, sons, texte, ...) dans un mail.

Depuis, le type MIME est une façon normalisée de classer les différents types de fichiers sur Internet. Les serveurs et les navigateurs Web disposent tous d'une liste de types MIME afin de pouvoir échanger des fichiers quelque soit leur type et le système d'exploitation utilisé. Pendant une transaction entre un serveur web et un navigateur internet, le serveur web envoie d'abord le type MIME du fichier au navigateur qui saura ainsi comment traiter et afficher le document.

Un type MIME comprend deux parties : un type et un sous-type séparé par une barre oblique (/).

Exemples

Fichier Type Sous-type Type MIME complet
Fichier HTML text html text/html
Fichier CSS text css text/css
Fichier GIF image gif image/gif
Fichier MP4 video mp4 video/mp4
Fichier MP3 audio mpeg audio/mpeg
