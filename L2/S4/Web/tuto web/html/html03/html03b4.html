    L'élément style est utilisé pour intégrer des règles de style qui s'appliqueront au contenu du document en cours.

    Les règles de styles sont écrites dans un langage spécifique : CCS (Cascading Style Sheet). Ce langage a sa propre grammaire et sa propre syntaxe. Il permet de mettre en page et de mettre en forme les éléments utilisés dans le corps du document. Un tutoriel est entièrement consacré à l'étude des CSS sans lesquelles un document HTML n'aurait pas du tout l'aspect que l'on voit sur le Web.

    Dans la réalité, comme nous l'avons vu à la page sur l'élément link les règles de style sont plutôt écrites et regroupées dans des fichiers CSS qui offrent le grand avantage de pouvoir être partagés par toutes ou partie des pages du site.

    Il n'en reste pas moins vrai que parfois des règles sont totalement spécifiques à une page, et dans ce cas elles pourront y être intégrées avec l'élément style comme le montre l'exemple ci-après.

    On peut utiliser un attribut type pour préciser le type MIME du code intégré. Dans le cas de règles de style CSS, la valeur de cet attribut doit être text/css. Cette valeur étant la valeur par défaut de l'élément, la présence de cet attribut est totalement inutile.
