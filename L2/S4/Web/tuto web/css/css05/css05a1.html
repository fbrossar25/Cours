Rappel sur les boîtes

    Un élément HTML est toujours affiché dans espace rectangulaire que l'on appelle une boîte.

    Suivant l'élément HTML la boîte d'affichage va s'empiler verticalement (boîte de type bloc) ou s'aligner horizontalement (boîte de type en ligne).

    Les boîtes sont construites avec différents composants, représentés par des propriétés CSS.

    Nous allons pouvoir donner des valeurs à ces propriérés pour modifier et individualiser la représentation graphique de la boîte.
    Les composants peuvent être regroupés en 4 familles :

    • agencement (marges externes margin, bordures border, marges internes padding)
    • dimensions (en particulier hauteur height et largeur width)
    • position (en particulier position, left, right, top, bottom, )
    • visuel (en particulier arrière plan background, police de caractères font, couleur de texte color).

    Cette partie du tutoriel concerne les propriétés d'agencement et de dimensionnement.

    La notion de boîte a été vues également dans Positionner des éléments -> C'est dans la boîte avec une description détaillée des types bloc et en ligne.
