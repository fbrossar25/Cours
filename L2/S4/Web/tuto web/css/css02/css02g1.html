Sélecteurs groupés

    Syntaxe S1, S2 

    dl, dt, dd,
ul, ol {
	margin: 0;
	padding: 0
	}

    Le combinateur virgule permet de définir le même bloc de décalarations de style pour tous les sélecteurs qu'il sépare. On parle aussi de regroupement de sélecteurs. Ce système est trés souvent utilisé car il permet de minimiser le code CSS et de rendre les modifications plus simples.

    Avec le combinateur virgule tous les sélecteurs de la règles sont concernés. C'est comme un opérateur ET.

    Les sélecteurs regroupés peuvent être de n'importe quel type ( simples, d'attributs, des pseudo classes, des pseudo éléments).

    L'exemple suivant entoure d'une bordure les éléments h3, h4 et ul. La couleur des caractères des éléments h3 et h4 sera rouge.
