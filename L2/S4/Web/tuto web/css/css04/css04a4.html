Graisse avec font-weight

    La propriété font-weight permet de mettre du texte en gras.

    Les spécifications définissent différentes valeurs possibles : normal, bold, bolder, lighter, 100, 200, 300, 400, 500, 600, 700, 800, 900, inherit.
    Dans la réalité seules 3 valeurs sont utilisables : bold, normal et inherit. inherit est la valeur par défaut car la propriété font-weight est héritée : elles est transmise automatiquement de l'élément parent à ses enfants, quelque soit le niveau de descendance.

    Italique avec font-style

    La propriété font-style permet de mettre du texte en italique. Toutes les polices n'ont pas une version italique ou oblique, en particulier les polices "exotiques" servant avant tout au titrage.

    Les spécifications définissent différentes valeurs possibles : normal, italic, oblique, inherit.
    Dans la réalité italic et oblique sont traitées de la même façon. inherit est la valeur par défaut car la propriété font-style est héritée : elles est transmise automatiquement de l'élément parent à ses enfants, quelque soit le niveau de descendance.

    Exemple : italique
    Fichier CSS : styles.css

    Petites majuscules avec font-variant

    La propriété font-variant change les caractères du texte. Pour le moment, la seule valeur ayant cet effet est small-caps qui met le texte en petites majuscules. C'est surtout intéressant pour le titrage.
