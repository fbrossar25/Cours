	\subsection{Exercice 2}
	On dispose des axiomes suivants :
	\begin{enumerate}
		\item $(X + Y) + Z = X + (Y + Z)$
		\item $e + X = X$
		\item $(-X) + X = e$
	\end{enumerate}
	Montrer que $\forall X, X + (-X) = e$ est un théorème déductif.\\
	Astuce :\\
	\begin{align*}
		X + (-X) &= --X + -X +X\\
		&= --X + (-X + X)\\
		&= --X + e\\
		\text{et } X + (-X) &= --X + -X +X\\
		&= e + X\\
		&= X\\
		\text{J'en déduis que } &X = --X + e\\ \\
		X + (-X) &= (--X + e) + (-X)\\
		&= --X + (e + -X)\\
		&= --X + -X\\
		&= e\\
	\end{align*}
